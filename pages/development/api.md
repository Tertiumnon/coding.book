# API

## Design

### Typical CRUD

- GET `/user` - get all users
- GET `/user/{id}` - get user with ID
- POST `/user` - create user
- PUT `/user/{id}` - update user with ID
- PATCH `/user/{id}` - patch user with ID
- DELETE `/user/{id}` - delete user with ID
